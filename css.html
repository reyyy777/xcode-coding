Kelas CSS

Pelajari cara mengatur tampilan halaman web dengan CSS

Pengantar CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya halaman HTML seperti warna, layout, font, dan lainnya.

Selektor CSS

Selektor digunakan untuk memilih elemen HTML yang ingin diberikan gaya. Beberapa contoh selektor:

  • p — memilih semua elemen paragraf
  • .kelas — memilih elemen dengan class tertentu
  • #id — memilih elemen dengan ID tertentu
  • div > p — memilih paragraf langsung di dalam div

Contoh Kode CSS

Berikut contoh dasar penulisan CSS:

body {
    font-family: Arial, sans-serif;
    background-color: #f9f9f9;
    color: #333;
}

h1 {
    color: #0066cc;
    text-align: center;
}

.kotak {
    width: 200px;
    height: 100px;
    background-color: lightblue;
    border: 1px solid #ccc;
}

Box Model CSS

Setiap elemen di HTML dianggap sebagai "kotak" yang terdiri dari:

  • Content → isi elemen (teks/gambar)
  • Padding → jarak antara content dengan border
  • Border → garis tepi elemen
  • Margin → jarak antar elemen

<style>
  .box {
    width: 200px;
    padding: 20px;
    border: 5px solid blue;
    margin: 15px;
    background-color: lightyellow;
  }
</style>

Warna & Background

CSS bisa mengatur warna dengan nama, HEX, RGB, atau HSL. Selain itu kita bisa memberi gambar sebagai background. 


<style>
  .warna1 { color: red; }
  .warna2 { color: #00ff00; }
  .warna3 { color: rgb(0,0,255); }
  .bg {
    background: linear-gradient(to right, orange, pink);
    padding: 15px;
    color: white;
  }
</style>

<p class="warna1">Teks merah</p>
<p class="warna2">Teks hijau (HEX)</p>
<p class="warna3">Teks biru (RGB)</p>
<div class="bg">Background gradasi warna</div>

Layout dengan Flexbox

Flexbox digunakan untuk mengatur layout elemen agar lebih mudah, misalnya membuat kolom sejajar. 


<style>
  .flex-container {
    display: flex;
    gap: 10px;
  }
  .flex-item {
    background: lightblue;
    padding: 20px;
    flex: 1;
    text-align: center;
  }
</style>

<div class="flex-container">
  <div class="flex-item">Kolom 1</div>
  <div class="flex-item">Kolom 2</div>
  <div class="flex-item">Kolom 3</div>
</div>
