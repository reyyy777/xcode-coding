Pengantar JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif. Dengan JavaScript, kita bisa membuat tombol interaktif, validasi form, animasi, dan banyak lagi.

1. Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di browser untuk membuat halaman web interaktif. 


<script>
  alert("Hello, JavaScript!");
</script>

2. Variabel & Tipe Data

JavaScript mendukung string, number, boolean, array, dan object.


let nama = "Riko";     // string
let umur = 20;         // number
let aktif = true;      // boolean

console.log(nama, umur, aktif);
Buka console (F12 â†’ tab Console) untuk melihat hasil dari console.log.

3. Operator & Percabangan


let nilai = 75;

if (nilai >= 80) {
  console.log("A");
} else if (nilai >= 60) {
  console.log("B");
} else {
  console.log("C");
}

4. Perulangan


for (let i = 1; i <= 5; i++) {
  console.log("Perulangan ke-" + i);
}

5. Fungsi


function sapa(nama) {
  return "Halo " + nama;
}

console.log(sapa("Riko"));

6. DOM Manipulation

JavaScript bisa mengubah isi HTML secara langsung.


<h3 id="judul">Judul Lama</h3>
<button onclick="ubahJudul()">Ubah Judul</button>

<script>
function ubahJudul() {
  document.getElementById("judul").innerText = "Judul Baru dengan JS!";
}
</script>
