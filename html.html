1. Pengenalan HTML

(HyperText Markup Language) adalah bahasa standar untuk membuat struktur dasar halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang memberi tahu browser bagaimana menampilkan teks, gambar, link, tabel, dll. Contoh sederhana: 


                <html>
                    <;head>
                      <&title>hello world</title>
                    </head>
                    <body>
                      <h1>hello world</h1>
                      <;p>this is my first website</p>
                    </body>
                 </html>

Struktur Dasar HTML

Berikut adalah contoh struktur dasar dari dokumen HTML:

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

Elemen HTML Umum

Beberapa elemen HTML yang sering digunakan:

  • <h1> hingga <h6> – Judul dengan tingkatan
  • <p> – Paragraf teks
  • <a> – Tautan/Link
  • <img> – Gambar
  • <ul>, <ol>, <li> – Daftar
  • <div>, <span> – Pembungkus konten

Heading, Paragraf, dan Teks

Heading (<h1> sampai <h6>) digunakan untuk membuat judul atau subjudul. <h1> = judul utama (paling besar) <h6> = judul terkecil Paragraf (<p>) digunakan untuk menulis teks biasa. Teks bisa diberi format seperti tebal (<b> / <strong>), miring (<i> / <em>), atau garis bawah (<u>). Contoh: 

<h1>Judul Utama</h1>
<h2>Subjudul</h2>
<p>Ini adalah sebuah paragraf. <b>Teks ini tebal</b>, sedangkan <i>teks ini miring</i>.</p>

5. Semantic HTML

Semantic HTML adalah elemen HTML yang punya arti/jelaskan fungsinya, bukan hanya gaya tampilan.

  • Membuat struktur web lebih jelas bagi browser & mesin pencari (SEO).
  • Membantu aksesibilitas (misalnya screen reader).

Contoh elemen semantic:

  • <header> = bagian atas halaman (judul, navigasi)
  • <nav> = navigasi menu
  • <main> = konten utama
  • <article> = artikel mandiri
  • <section> = bagian dari konten
  • <footer> = bagian bawah halaman

            <header>
  <h1>Blog Pribadi</h1>
  <nav>
    <a href="index.html">Home</a> | <a href="about.html">Tentang</a>
  </nav>
</header>

<main>
  <article>
    <h2>Judul Artikel</h2>
    <p>Isi artikel...</p>
  </article>
</main>

<footer>
  <p>© 2025 Blogku</p>
</footer>
